## Tech Stack
- **React 18** - UI framework
- **Vite** - Build tool and dev server
- **React Router** - Client-side routing
- **GitHub Actions** - CI/CD and deployment

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The site is configured to deploy to GitHub Pages using GitHub Actions. The workflow automatically builds and deploys on push to main branch.
